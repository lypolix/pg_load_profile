# PG Load Profile - Frontend

React-приложение для мониторинга производительности PostgreSQL базы данных.

## 🚀 Начало работы

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm start
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для production

```bash
npm build
```

## ✨ Особенности

- 🎨 **Анимированный градиентный фон** - плавно переливающийся фон на всех страницах
- 🔒 **Аутентификация** - система входа с защищенными маршрутами
- 📊 **Dashboard** - панель мониторинга с реальными метриками в реальном времени
- 💅 **Современный UI** - построен с помощью Tailwind CSS
- ⚡ **React Router** - клиентская маршрутизация
- 📱 **Адаптивный дизайн** - работает на различных размерах экранов

## 🛠️ Технологии

- React 18
- TypeScript
- React Router v6
- Tailwind CSS
- CSS Animations

## 📋 Доступные страницы

1. **Login** (`/login`) - Страница входа с анимированным фоном
2. **Dashboard** (`/dashboard`) - Главная панель управления (требует аутентификации)

## 🔄 Обновления в реальном времени

Dashboard обновляет статистику каждые 3 секунды, симулируя реальные данные от базы данных.
